# Data Dictionary

| Column | Type | Description |
|---|---|---|
| satisfaction_level | Numeric | Employee satisfaction level from 0 to 1 |
| last_evaluation | Numeric | Last evaluation score from 0 to 1 |
| number_project | Integer | Number of projects assigned |
| average_monthly_hours | Integer | Average monthly working hours |
| time_spend_company | Integer | Years spent at the company |
| Work_accident | Binary | 1 if the employee had a work accident, else 0 |
| left | Binary | 1 if the employee left, else 0 |
| promotion_last_5years | Binary | 1 if promoted in last 5 years, else 0 |
| department | Categorical | Employee department |
| salary | Categorical | Salary level: low, medium, high |
